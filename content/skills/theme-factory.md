## What This Skill Does

Provides a curated collection of 10 professional themes, each with carefully selected color palettes and font pairings for consistent, polished artifacts.

<Card title="Core Capabilities">

- **10 Pre-set Themes** — Professional color palettes and font pairings
- **Theme Showcase** — Visual preview via theme-showcase.pdf
- **Apply to Any Artifact** — Slides, docs, HTML landing pages, reports
- **Custom Theme Creation** — Generate new themes on-the-fly based on requirements
- **Consistent Styling** — Cohesive visual identity across all slides/pages

</Card>

## Usage

### Apply a Theme

```

```
Apply the Ocean Depths theme to this presentation
```

**User**: "Style this slide deck with a professional theme"

**Process**:
```
1. Display theme-showcase.pdf for visual reference

4. Apply chosen theme's colors and fonts consistently

Applied to:
- All slide backgrounds
- Title and heading styles
- Body text formatting
- Chart and graph colors
- Footer and navigation elements

## Application Process

1. Show the `theme-showcase.pdf` file for visual reference
2. Ask which theme to apply to the deck/artifact
3. Wait for explicit confirmation of chosen theme
4. Apply the selected theme's colors and fonts consistently
5. Ensure proper contrast and readability
6. Maintain the theme's visual identity across all elements

## Creating Custom Themes

When none of the 10 preset themes fit:

```
Create a custom theme for [context] — [description of desired feel]
```

**Example**:
```
Create a custom theme for a children's education app — playful, colorful, friendly
```

**Output**:
- Custom theme name reflecting the concept
- Appropriate color palette (bright, friendly colors)
- Font pairings (rounded, approachable typefaces)
- Preview for review before applying

## Tips for Best Results

1. **Match theme to audience** — Corporate vs. creative vs. educational
2. **Consider accessibility** — Ensure sufficient contrast
3. **Stay consistent** — Use one theme throughout the artifact
4. **Test readability** — Verify text is legible on all backgrounds
5. **Export showcase** — Keep theme-showcase.pdf for reference
