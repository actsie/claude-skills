# GSAP Skills

Official AI skills for GSAP. These skills teach Claude, Cursor, Copilot, and other coding agents how to use GreenSock Animation Platform correctly, including core animation patterns, timelines, ScrollTrigger, framework usage, performance guidance, and plugin-specific workflows.

## What It Includes

- **GSAP Core** — guidance for tweens, timelines, easing, staggers, defaults, and common animation setup
- **ScrollTrigger** — patterns for scroll-linked animation, pinning, scrub behavior, and trigger configuration
- **Framework Usage** — React, Vue, Svelte, and lifecycle-safe animation patterns
- **Plugin Workflows** — references for MotionPath, Flip, Draggable, Observer, ScrollSmoother, and other GSAP plugins
- **Performance Guidance** — practical rules for transform-based animation, batching, and avoiding layout thrashing
